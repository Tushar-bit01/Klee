# ArcaneVerse

ArcaneVerse is a frontend-focused project inspired by the *Arcane* series. The goal of this project was to improve frontend development skills by building an interactive and animation-rich website that prioritizes smooth user experience over basic functionality.

---

## Live Demo and Repository

- Live Application: https://arcaneeee.netlify.app/

---

## Project Overview

This project was built to explore advanced frontend concepts such as animations, page transitions, and interactive UI elements. Instead of focusing solely on static layouts, the emphasis was on creating a smooth, immersive, and engaging user experience.

---

## Features

- Animated navigation bar with a stair-style open and close effect  
- Page transition animations when navigating between routes  
- Home page with background music and a sound toggle  
- Songs page with playable Arcane soundtrack  
- Characters page featuring scroll-based animations using GSAP ScrollTrigger  
- Interactive spinning character selector  
- Multi-language support (English and French)  
- Fully responsive layout across all device sizes  

---

## Tech Stack

- React  
- Tailwind CSS  
- React Router  
- GSAP (GreenSock Animation Platform)

---

## What I Learned

- Implementing complex animations using GSAP  
- Managing smooth page transitions in a single-page application  
- Improving user interaction through motion and visual feedback  
- Structuring a scalable frontend project with React and modern tooling
